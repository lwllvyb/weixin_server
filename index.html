FAQ 常见问题
和别的 IP 库有什么区别？
1、我们会更严格的对 IP 数据进行审核入库，我们对准确度的要求高于精准度的要求；
2、我们通过总结的一些方法通过程序或人工方式进行 IP 数据的收集分析与审核入库，包括但不限于：
(1) 我们在全球的 280+ 个监测点，不断增加中；
(2) 自有或者其它可信任的数据来源；
(3) 全球 WHOIS/BGP/ASN 数据；
(4) 全球 Internet Exchange Point 数据（已在 IP 数据库中）；
(5) 自行维护的全球 rDNS 以及骨干网路由 IP 数据库；
(6) 客户、合作伙伴的反馈数据；
(7) 网友提交纠错的数据；
(8) 其它方式。
另外，请您一定注意，我们强调的是“我们对准确度的要求高于精准度的要求”，请仔细思考准确度和精准度的区别。
IP 库的准确度如何？
如果存在一个 100% 准确的 IP 库可以进行对比的话，我也直接去买了。
何况实际上，全球 IP 的数据每天都有大量的变更，所以理论上也不可能存在一个 100% 准确的 IP 库的。
在没有所谓标准答案的情况下，比的只能是谁的办法多和可以拿到的相关数据多了。请参见上一个问题。
